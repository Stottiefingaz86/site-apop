## Summary

Rename the "Casino" navigation label to "Games" and "Live Casino" to "Live Games" across all user-facing navigation touchpoints site-wide. This is a display-label-only change — no URLs, routes, internal identifiers, or analytics slugs are modified. The navigation is currently duplicated across ~73 page files (not a shared component), so each file must be updated individually.

## Change Patterns

The following string replacement patterns cover all navigation label instances:

### Pattern 1: Navigation array label property
```typescript
// Before:
{ label: 'Casino', page: 'casino' }
{ label: 'Live Casino', page: 'liveCasino' }

// After:
{ label: 'Games', page: 'casino' }
{ label: 'Live Games', page: 'liveCasino' }
```

### Pattern 2: Quick-link array label property
```typescript
// Before:
{ label: 'Casino', onClick: () => { ... } }
{ label: 'Live Casino', onClick: () => { ... } }

// After:
{ label: 'Games', onClick: () => { ... } }
{ label: 'Live Games', onClick: () => { ... } }
```

### Pattern 3: Desktop SidebarMenuButton text content
```typescript
// Before:
<span className="relative z-10">Casino</span>
<span className="relative z-10">Live Casino</span>

// After:
<span className="relative z-10">Games</span>
<span className="relative z-10">Live Games</span>
```

### Pattern 4: Conditional label checks
```typescript
// Before:
item.label === 'Casino'

// After:
item.label === 'Games'
```

### Pattern 5: trackNav display name parameter (second argument)
```typescript
// Before:
trackNav('casino', 'Casino')
trackNav('casino', 'Live Casino')

// After:
trackNav('casino', 'Games')
trackNav('casino', 'Live Games')
```

### Pattern 6: trackPageView display name parameter (second argument)
```typescript
// Before:
trackPageView('casino', 'Casino')
trackPageView('live-casino', 'Live Casino')

// After:
trackPageView('casino', 'Games')
trackPageView('live-casino', 'Live Games')
```

### Pattern 7: Design customizer product label
```typescript
// Before:
{ key: 'casino', label: 'Casino' }

// After:
{ key: 'casino', label: 'Games' }
```

## What Does NOT Change

- URL paths: `/casino`, `/casino?tab=live`, `/casino?live=true`, `/casino?poker=true`, `/casino?vip=true`
- Internal identifiers: `page: 'casino'`, `product: 'casino'`, `visibleProducts.casino`
- Analytics slugs: `trackNav('casino', ...)` first argument, `trackPageView('casino', ...)` first argument
- Animation IDs: `layoutId="casinoNavPill"`
- Chat room identifiers: `activeRoom === 'casino'`
- Sidebar menu items that are not navigation labels (e.g., "Live Casino" as a game category in sidebar lists — these refer to the game type, not navigation)
